PEMBAYARAN KULIAH MELALUI BANK BNI
Pembayaran melalui BNI atm :
1. Menu pembayaran
2. Biaya pendidikan
3. Spc
4. Pilih 8003 (kode ITS)
5. Masukkan Kode Bayar (91+No Pendaftaran) untuk Mahasiswa Baru dan Kode Bayar
(91+NRP) untuk Mahasiswa Lama
Pembayaran melalui BNI Mobile banking:
6. Menu pembayaran
7. Pilih biaya pendidikan
8. Jenis layanan pilih pembayaran
9. Pilih perguruan tinggi (ITS)
10. Input no billing Kode Bayar (91+No Pendaftaran) dan Kode Bayar (91+NRP) untuk
Mahasiswa Lama
Pembayaran melalui BNI Internet banking:
11. Menu pembayaran
12. Biaya pendidikan
13. Tipe layanan pilih pembayaran
14. Pilih perguruan tinggi (ITS)
15. Input no tagihan Masukkan (91+No Pendaftaran) dan Kode Bayar (91+NRP) untuk
Mahasiswa Lama
Pembayaran melalui Teller BNI
Silahkan menghubung teller Bank BNI terdekat
